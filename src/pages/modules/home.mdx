<PageDescription>

The **<SolutionName />** **Home** module enables organizations to view
statistical usage information. The number of users, products, and facilities in
your organization are displayed, along with trace usage, user logins and event uploads.

</PageDescription>

## How it works

The **Home** module compiles and displays current data for your organization based
on user logins and data uploads. Activity by recent time periods are displayed by default,
enabling authorized users to view a snapshot of <SolutionName /> usage trends for their organization.
